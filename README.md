**RecipeJS** is a functional cooking companion that helps you explore delicious recipes with ease. Built with vanilla JavaScript, it demonstrates dynamic DOM manipulation and functional programming concepts.

## Features
- **Dynamic Recipe Cards**: Automatically generated from a data array.
- **Difficulty Badges**: Visual indicators for Easy, Medium, and Hard recipes.
- **Responsive Design**: Looks great on desktop and mobile.
- **Interactive UI**: smooth hover effects and shadows.

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
